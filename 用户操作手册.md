# 素珍管理系统 - 用户操作手册

## 快速开始

### 安装部署

1. 安装 [Docker Desktop](https://www.docker.com/products/docker-desktop/) 并重启电脑
2. 解压交付包，双击 **`setup.bat`**，按提示完成配置
3. 浏览器访问 `http://localhost:端口号`
4. 初始账号：`admin` / 密码：`admin!`（首次登录后务必修改密码）

### 停止与启动

- 双击 **`stop.bat`** 停止服务
- 再次双击 **`setup.bat`** 重新启动

## 常用操作

### 搜索与筛选

每个管理页面顶部有筛选栏，选择条件后点击 **🔍 搜索** 按钮或按 Enter 键查询。

### 列显示配置

点击表格上方的 **⚙️ 列显示** 按钮，勾选需要显示的列，配置自动保存到浏览器。

### 批量操作

勾选表格行复选框，使用批量操作栏一键设置邮件提醒或重要等级。

### 邮件提醒

- 编辑记录时勾选"邮件提醒"复选框
- 在 **系统配置** 中配置 SMTP 邮箱和提醒时间
- 到期前指定天数内的记录会自动发送提醒邮件

### 本地文件同步

点击 **📂 本地文件同步** 按钮：
- 输入目录路径（Docker环境用容器内路径如 `/data/合同文件`）
- 点击 📂 按钮选择文件夹自动填入路径
- 勾选需要导入的文件，点击导入

### 数据导出

进入 **数据导出** 页面，选择导出方式和日期范围，点击导出生成 Excel 文件。

## 常见问题

**Q: 上传文件没反应？**
文件类别固定为 合同/专利/车险续期，上传时会提示选择。

**Q: 邮件测试成功但自动提醒不发送？**
检查系统配置中"启用到期提醒"是否开启，以及提醒时间和天数设置。

**Q: Docker 环境下扫描不到本地文件夹？**
需先将文件夹放入挂载目录，容器内路径为 `/data/子文件夹名`。
